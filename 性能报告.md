## 运行方式

1. 在 `TestProject/Packages/manifest.json` 中加入 UniTask。
2. 在 Unity Player Settings 中加入 `TASK233_HAS_UNITASK`。
3. 配置 GitHub secrets：`UNITY_LICENSE` 或 `UNITY_SERIAL`。
4. 运行 GitHub Actions：`Unity Performance Tests`。

也可以本地运行：

```powershell
Unity.exe -batchmode -projectPath TestProject -runTests -testPlatform EditMode -testResults artifacts/editmode-results.xml
```
